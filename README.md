О проекте:

Приложение для автоматизации закупок в розничной сети – этот сервис предоставляет REST API для взаимодействия покупателя, поставщика и администратора.

Целью проекта:

Является создание сервис для автоматизации закупок, который позволяет:

Покупателям просматривать каталог товаров от разных поставщиков и оформлять заказы.

Поставщикам управлять своим прайс-листом через импорт файлов.

Администраторам контролировать весь процесс закупок.

Основные возможности:

Для покупателей:

1. Регистрация и аутентификация

2. Просмотр каталога товаров с фильтрацией по категориям

3. Управление корзиной (добавление/удаление товаров)

4. Оформление заказов с указанием адреса доставки

5. Получение email-подтверждений

Для поставщиков:

1. Импорт товаров через YAML файлы

2. Управление своими товарами (цена, количество)

3. Просмотр заказов на свои товары

Для администраторов:

1. Управление пользователями и поставщиками

2. Управление категориями и товарами

3. Просмотр и управление всеми заказами

Общие:

1. Интерактивная документация API

2. Автоматические email-уведомления

3. Контейнеризация с Docker

4. Асинхронные задачи через Celery


Предварительные требования для запуска проекта:

Python 3.11+

Redis (для Celery)

Docker и Docker Compose (опционально)
